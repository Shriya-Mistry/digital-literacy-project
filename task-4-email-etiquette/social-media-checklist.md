# SOCIAL MEDIA CHECKLIST (DO'S AND DON'TS)


| Do's | Don'ts |
|------|--------|
| Verify before sharing any information | Restrain from spreading fake news |
| Comments on social media should be respectful | Engagement in online argument should not be done |
| Screen time per day should be limited | Don't use social media for validation (likes/followers) |
| Social Media should be used for growth, by connecting to people and learning new skills | Personal details should not be overshared |
| Use strong passwords | Don't ignore account security |
